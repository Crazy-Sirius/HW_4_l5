Селекторы

По тегам

h2 {
color: red;
}
Применится к заголовку h1 и изменит его цвет на красный

По классам

.octopus {
color: green;
}
Применится ко всем объектам, у которых class = octopus

По ID

#MainButton {
color: blue;
}
Применится к единственному элементу, ID которого MainButton

По нескольким классам

.Bot.Text {
color: brown;
}
Стили будут применяться ко всем элементам, которые одновременно имеют и класс MainButton и класс NewPage
